# Task list - a CLI tool for tracking your tasks, sorted by categories.

## How to run:
python3 tracker.py 

## How to use:
add - command for adding anything
del - delete any task or category 
done - marks a task as completed
back/stop - returns to main menu, or shut down the CLI


## Features:
save all your progress in JSON between session, so you won't lose any of your tasks
count your real time of work, you can start stopwatch for every task
